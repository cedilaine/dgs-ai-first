# dgs-ai-first

Repositório para centralizar os entregáveis da trilha de IA, com foco em cenários, análises e artefatos de apoio produzidos ao longo dos exercícios.

## Estrutura

- `docs/cenarios/`: cenários e entregáveis organizados por arquivo.

## Conteúdo atual

- `docs/cenarios/cenario-1.md`: exercícios, cenários de falha, rubrica de avaliação e plano de testes para o pipeline de RAG.

## Convenção sugerida

- Criar novos cenários no formato `docs/cenarios/cenario-N.md`.
- Manter cada arquivo com contexto, tarefa, análise e entregáveis no mesmo documento.
- Preservar referências explícitas aos documentos-fonte quando houver análise de políticas, procedimentos ou SLAs.

## Próximos passos

- Adicionar novos cenários conforme a evolução da trilha.
- Separar, se necessário, artefatos complementares em pastas dedicadas como `docs/rubricas/` ou `docs/planos-de-teste/`.
